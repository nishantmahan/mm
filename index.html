FAQ and privacy

Trust basics for a paid tool.

Can I export my data?

Yes. The app exports JSON for backup and CSV for spreadsheets.

Does it work like an app?

Yes. It includes PWA manifest and service worker files for mobile and desktop install.

Is login included?

Not yet. The current version is frontend-first. Add Firebase Auth before taking payments.

Privacy policy

DailyLink stores task groups, URLs, titles, notes, due dates, and completion states in Firebase Realtime Database. Browser notification permission stays in the browser. Before selling, add per-user authentication and security rules so each customer can only read and write their own data.
